基于短信的发送及点击数据,根据用户手机号码特征和购买物品的二级类别来计算对用户点击短信链接的概率
特征:province、head、tail、keyword

模型的有效性评估需考虑的方面
1.业务场景,看是否有利可图
2.实际测试,看点击率的提升度

线上部署脚本
V1.0-原始版:将uid的所有特征数据UNION在一张表,然后计算其总的线性权重得分,最后计算逻辑回归的得分
V1.1-改进版:分别计算uid手机号码特征和关键字特征的权重得分,再UNION在一张表并计算uid总的线性权重得分,最后计算逻辑回归的得分
说明:相比V1.0的UNION,V1.1的UNION大大减少了数据量,因此运行效率更高
